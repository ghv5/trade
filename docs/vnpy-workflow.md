# vn.py 工作流

当前阶段先完成本地投研闭环，不接实盘接口。

## 阶段

1. 安装独立 Python 环境和 vn.py 插件。
2. 将通达信日线导入 vn.py SQLite 数据库。
3. 在 DataManager 中检查历史数据完整性。
4. 使用 CTA Backtester 做单标的策略回测。
5. 稳定后再接入模拟交易和实盘接口。

## 本地数据约定

- 行情原始文件放入 `data/` 或 `tdx/`，不提交 Git。
- vn.py 默认运行目录 `.vntrader/` 不提交 Git。
- SQLite 数据库文件不提交 Git。

## 风险边界

现阶段只做数据导入、回测和模拟交易准备。任何实盘接口、账号密钥和资金相关配置都必须放在本地环境或密钥管理系统中，不写入仓库。
